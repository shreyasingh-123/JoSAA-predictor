# JOSAA Branch Predictor

Python CLI tool using JOSAA 2025 Round 6 cutoffs to predict NIT/IIIT branch allocation.

**Built to solve my own counseling dilemma**: NIT options  

### Features
- Home State vs Other State logic for different NITs
- Gender-based cutoff handling 
- Dynamic search across all colleges
- Fallback logic for missing data

### Run it
`python josaa_predictor.py`

**Note**: Data from JOSAA 2025 Round 6. OBC/EWS/SC/ST not added yet.
